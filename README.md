# 🌱 GROOT Recon Tool

A powerful automated reconnaissance framework for penetration testers and security researchers

## Installation
git clone https://github.com/chiragt1-art/groot-recon.git
cd groot-recon
chmod +x recon.sh
./recon.sh

- 🌍 DNS Recon
